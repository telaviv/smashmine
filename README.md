# smashmine

A stastistics site for smash bros wii u tournaments.

## Start

```sh
$ npm install
$ npm start
```

You can change server port using following command.

```sh
$ PORT=5000 npm start
```

## Testing

```sh
$ npm test
```

note: jsdom >= v4.0 can not work on node 0.x
